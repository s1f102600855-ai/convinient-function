Q1

表示される文字は？

次のコードを実行した結果を選んでください。

name = "Python"

Q2

計算結果は？

掛け算の結果として表示される値を選んでください。

price = 120
count = 3
print(price * count)

答えを選んでください。

Q3

if文の結果は？

条件に当てはまるとき、どちらが表示されるでしょうか。

score = 80
if score >= 60:
    print("合格")
else:
    print("もう一度練習")

答えを選んでください。

Q4

リストの合計は？

sumで計算される合計を選んでください。

scores = [70, 80, 90]
print(sum(scores))

答えを選んでください。

Q5

答えを選んでください。
