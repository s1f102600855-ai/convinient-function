基礎の順番

まずは、短いコードを動かしながら1つずつ確認するのがおすすめです。

01

変数とprint

文字や数字を変数に入れて、printで画面に表示します。

name = "Python"
print(name)

02

ifで条件分岐

条件に合わせて、実行する処理を切り替えます。

score = 80
if score >= 60:
    print("合格")

03

forで繰り返し

同じような処理を、リストや回数に合わせて繰り返します。

for number in range(3):
    print(number)

04

リストと計算

複数の値をまとめ、合計や個数を計算します。

scores = [70, 80, 90]
print(sum(scores))

05

関数を作る

何度も使う処理を、名前をつけてまとめます。

def greet(name):
    return f"こんにちは、{name}さん"
