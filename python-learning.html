基礎の順番

まずは、短いコードを動かし、数字や文字を少し変えながら1つずつ確認するのがおすすめです。

01

変数とprint

文字や数字を変数に入れて、printで画面に表示します。

name = "Python"
print(name)

02

ifで条件分岐

条件に合わせて、実行する処理を切り替えます。

score = 80
if score >= 60:
    print("合格")

03

forで繰り返し

同じような処理を、リストや回数に合わせて繰り返します。

for number in range(3):
    print(number)

04

リストと計算

複数の値をまとめ、合計や個数を計算します。

scores = [70, 80, 90]
print(sum(scores))

05

関数を作る

何度も使う処理を、名前をつけてまとめます。

def greet(name):
    return f"こんにちは、{name}さん"

06

四則演算

足し算、引き算、掛け算、割り算を使い、計算結果を表示します。

price = 120
count = 3
print(price * count)

07

文字列とf-string

文字と変数を組み合わせて、読みやすい文章を作ります。

item = "ノート"
print(f"{item}を買いました")

08

比較と論理演算

複数の条件を組み合わせて、より細かく判定します。

age = 20
has_ticket = True

09

リストに追加

あとから値を追加し、リストの中身を順番に表示します。

tasks = []
tasks.append("メモを書く")

10

辞書で管理

名前つきのデータをまとめ、キーを使って値を取り出します。

profile = {"name": "Aoi"}
print(profile["name"])

11

whileで繰り返し

条件が成り立つ間だけ、同じ処理を繰り返します。

count = 3
while count > 0:
    print(count)

12

enumerateで番号つき表示

リストの順番と値を一緒に取り出して表示します。

tools = ["メモ", "カレンダー"]
for index, tool in enumerate(tools):
